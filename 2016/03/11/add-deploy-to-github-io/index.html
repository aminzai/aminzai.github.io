部屬 Hexo 到 github.io

在以前寫靜態網頁都要另外找網頁空間來做hosting，至從Github.io出來以後就可以直接用Github.io進行hosting。
Hexo好處是可以直接部屬編譯完後的static檔案直接傳至github上，不用把整個Hexo都上傳至github上。

先申請github.io

申請網址：https://pages.github.com/

安裝hexo plugin

加入部屬設定到_config.yml上

參數敘述
repo（必要）github.io的repo位置
branch（選項）要上傳的branch
message（選項）上傳後的commit訊息 預設：Site updated: YYYY-MM-DD HH:mm:ss
